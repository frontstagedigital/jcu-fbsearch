# FBSearch - JCU SSJS

Each folder contains a single JS file named after the folder.
Load them in this order to reproduce the full bundle:

1. utils/utils.js
2. api/api.js
3. facets/facets.js
4. data/data.js
5. render/render.js
6. page/page.js
7. zero/zero.js
8. diagnostics/diagnostics.js   (optional)
9. init/init.js
